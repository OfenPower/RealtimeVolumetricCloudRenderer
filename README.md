# RealtimeVolumetricCloudRenderer

This is a demo application which renders volumetric clouds in realtime, combined with the preetham atmospheric scattering model. 
It comes with an ImGUI UI Interface to control the rendering process such as cloud model-, cloud lighting-, raymarch- and wind settings.
